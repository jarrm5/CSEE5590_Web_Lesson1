CSEE5590: Web Lesson 1

Creating a github repo is very simple using the git terminal.

The steps to do it are:

  1. Create the repo on your github homepage
  2. Clone the repo to your local repository
    3. init repo locally demo
  3. Add a README.md file
  4. Set the README.md file for staging
  5. Commit the file with a message
    6. First commit demo
  6. Add the file to the remote repo
    7. second commit demo
  7. Do a git pull to sync the remote with the local repo
  8. Do a git status to make sure the local repo is up-to-date with the remote repo

Github Tips for success

  • Try to do github actions through the terminal - it will reinforce your understanding of how it works.
  • Create a test repo and try to break it (create a merge conflict and try to solve it)
  • Experiment with all of github's features (forking, pull requests, etc.)
